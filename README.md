## Job Application's Status Tracker
----
**Description** <br>
This is a Full Stack (MERN) Job Portal Application built with 
- React for Front End
- Redux-Thunk for Asynchronous operations
- Node JS for Runtime
- Express JS for Backend Routing
- Mongo DB for Database
